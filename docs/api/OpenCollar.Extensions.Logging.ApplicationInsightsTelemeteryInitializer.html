    Class SqlClientTelemeteryInitializer

    A telemetry initializer that adds the contextual information captured by the OpenCollar.Extensions.Logging.LoggingContext into the custom data recorded by the SqlClient logger. This makes it easier to search and filter by these values in the online tools.
